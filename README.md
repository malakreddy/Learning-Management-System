# LMS - Learning Management System

A minimalist, content-first Learning Management System built with Next.js and Express.

## Project Structure

- `backend/`: Express server with Prisma and MySQL.
- `frontend/`: Next.js application with Tailwind CSS.

## Getting Started

1. Install dependencies in the root:
   ```bash
   npm run install-all
   ```
2. Set up environment variables in `backend/.env` and `frontend/.env.local`.
3. Run the project:
   ```bash
   npm start
   ```
